# 🌐 Personal Portfolio Website

A modern, responsive portfolio website built with **React** to showcase my projects, technical skills, experience, achievements, and community involvement.

## 🚀 About

This portfolio highlights my journey as a Computer Science Engineering student, featuring:

* Full-stack and frontend development projects
* Technical skills and technologies
* Academic achievements
* Volunteering and leadership experiences
* Community contributions through GDG On Campus, hackathons, and tech events

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite
* Lucide React Icons

## ✨ Features

* Responsive design for desktop and mobile devices
* Interactive project showcase
* Skills and achievements section

## 🔧 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 📦 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
